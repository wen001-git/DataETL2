## 优先级说明

| 标识 | 含义 |
|------|------|
| **M** | Must Have — MVP 必须交付，缺失则产品无法使用 |

## 模块 A — 数据源连接管理 (Source Connector)

> Phase 1 仅支持 SFTP 拉取和手动文件上传两种接入方式，其余接入类型延至后续版本。

| 功能 ID | 功能名称 | 描述 | 优先级 | Phase |
|---------|---------|------|-------|-------|
| A-01 | 数据源配置向导 | 引导用户通过表单配置数据源（名称/接入类型/目标层级/目标表名，全部可配置） | **M** | 1 |
| A-02 | 手动文件上传 | 在浏览器直接上传 CSV、Excel (.xlsx/.xls) 文件，DuckDB 解析后入库 etl_raw | **M** | 1 |
| A-03 | SFTP 连接 | 配置 SFTP 连接参数（host/port/user/password/path/file_pattern），凭据加密存储 | **M** | 1 |
| A-04 | SFTP 文件浏览与拉取 | 列出 SFTP 目录文件，手动选择文件拉取入库 etl_raw | **M** | 1 |
| A-05 | 字段预览 | 文件上传或拉取后，显示字段名、类型和前 10 行样本数据 | **M** | 1 |
| A-06 | 连接测试与诊断 | 一键测试 SFTP 连通性，显示详细错误诊断信息 | **M** | 1 |
| A-07 | 凭据加密存储 | SFTP 密码使用 Fernet 加密存储，API 响应不返回明文 | **M** | 1 |
| A-08 | 数据源列表/搜索 | 列出、搜索、管理已配置的数据源 | **S** | 1 |
| A-09 | 关系型数据库连接 | 支持 MySQL、PostgreSQL、SQL Server、Oracle、SQLite | **S** | 2 |
| A-10 | REST API 连接 | 通过 URL + 认证配置拉取 HTTP/HTTPS API 数据 | **N** | 2 |
| A-11 | 云存储连接 | 支持 AWS S3、Azure Blob、Google Cloud Storage | **N** | 2 |
| A-12 | 消息队列连接 | 支持 Kafka、RabbitMQ 实时流数据接入 | **N** | 3 |

## 模块 B — 管道配置管理 (Pipeline Configuration)

> Phase 1 使用表单式 UI（非拖拽画布）完成数据源→字段映射→过滤规则→执行的全链路配置；可视化拖拽画布在 Phase 2 引入。

| 功能 ID | 功能名称 | 描述 | 优先级 | Phase |
|---------|---------|------|-------|-------|
| B-01 | 字段映射编辑器 | 左列源字段（Raw）对应右列目标字段（DWD），支持重命名/类型转换/默认值/跳过 | **M** | 1 |
| B-02 | 批量导入字段映射 | 上传 Excel 映射模板，系统解析为字段映射规则批量入库 | **M** | 1 |
| B-03 | 手动单行编辑映射 | 在 UI 中逐条新增/编辑/删除字段映射规则 | **M** | 1 |
| B-04 | 过滤规则配置 | 配置行过滤条件（字段 + 运算符 + 值，AND/OR 逻辑组合） | **M** | 1 |
| B-05 | DWS 聚合规则配置 | 配置 GROUP BY 字段和聚合函数（SUM/AVG/COUNT/MAX/MIN），目标表名用户自定义 | **M** | 1 |
| B-06 | ADS 输出规则配置 | 配置字段选取、排序、条件过滤，目标表名用户自定义 | **M** | 1 |
| B-07 | 配置保存 | 数据源 + 字段映射 + 过滤规则持久化保存，可重复执行 | **M** | 1 |
| B-08 | 可视化拖拽画布 | 拖拽节点、连线，构建 ETL 数据流有向图（React Flow） | **S** | 2 |
| B-09 | 节点库面板 | 分类展示所有可用节点（源节点/转换节点/目标节点） | **S** | 2 |
| B-10 | 管道导入/导出 | 将配置导出为 JSON/YAML，支持跨环境迁移 | **S** | 2 |
| B-11 | 管道模板库 | 内置常用场景模板 | **N** | 3 |

## 模块 C — 数据转换节点 (Transformation Nodes)

| 功能 ID | 功能名称 | 描述 | 优先级 |
|---------|---------|------|-------|
| C-01 | 字段映射 | 源字段到目标字段的名称/类型映射配置 | **M** |
| C-02 | 数据类型转换 | 字符串/数字/日期/布尔类型互转 | **M** |
| C-03 | 行过滤 (Filter) | 基于条件表达式筛选行（支持 AND/OR 逻辑组合） | **M** |
| C-04 | 列选择/删除 | 选择保留或删除指定列 | **M** |
| C-05 | 字段计算 (Formula) | 基于已有字段计算新字段（支持常用函数） | **M** |
| C-06 | 聚合/分组 (Aggregate) | GROUP BY + 聚合函数（SUM/AVG/COUNT/MAX/MIN） | **M** |
| C-07 | 数据关联 (Join) | 支持 INNER/LEFT/RIGHT/FULL JOIN，多数据源合并 | **M** |
| C-08 | 数据去重 | 基于指定键字段去重，保留最新/最旧记录 | **S** |
| C-09 | 空值处理 | 填充默认值、删除空值行、标记空值列 | **S** |
| C-10 | 字符串处理 | 截取、拼接、正则提取、大小写转换、Trim | **S** |
| C-11 | 日期格式化 | 日期解析与格式统一化（支持多种格式输入） | **S** |
| C-12 | 数据排序 | 按单列或多列升序/降序排列 | **S** |
| C-13 | 数据抽样 | 按比例或行数随机抽样 | **N** |
| C-14 | 透视/逆透视 | Pivot/Unpivot 行列转换 | **N** |
| C-15 | 自定义 SQL 转换 | 对节点数据执行用户输入的 SQL 语句 | **S** |
| C-16 | 自定义 Python 转换 | 执行用户 Python 代码（沙箱隔离执行） | **N** |
| C-17 | 数据验证节点 | 配置规则校验数据质量，不合格行路由到错误输出 | **S** |
| C-18 | 数据脱敏 | 手机号/邮箱/身份证等敏感字段掩码处理 | **S** |

## 模块 D — 数据目标写入 (Destination Connector)

| 功能 ID | 功能名称 | 描述 | 优先级 |
|---------|---------|------|-------|
| D-01 | 关系型数据库写入 | INSERT / UPSERT / REPLACE 三种写入模式 | **M** |
| D-02 | 文件下载 | 将结果导出为 CSV、Excel、JSON 文件下载 | **M** |
| D-03 | 云存储写入 | 写入 AWS S3 / Azure Blob / GCS | **S** |
| D-04 | SFTP 写出 | 将文件推送到 SFTP 服务器 | **S** |
| D-05 | REST API 推送 | 将数据 POST 到外部 HTTP 接口 | **S** |
| D-06 | Webhook 通知 | 执行完成后触发外部 Webhook 回调 | **S** |
| D-07 | 邮件报告发送 | 将结果以 Excel 附件通过邮件发送 | **N** |
| D-08 | 分区写入 | 按日期/字段值自动分区写出文件 | **N** |

## 模块 E — 调度与执行 (Scheduling & Execution)

| 功能 ID | 功能名称 | 描述 | 优先级 | Phase |
|---------|---------|------|-------|-------|
| E-01 | 手动触发执行 | 一键立即运行完整链路（Raw→DWD→DWS→ADS）或指定层级 | **M** | 1 |
| E-02 | 执行层级选择 | 触发时可选择执行全链路或指定起止层（如仅 DWD→DWS） | **M** | 1 |
| E-03 | Cron 表达式调度 | 配置 Cron 表达式定时执行（Prefect 3.x 调度） | **S** | 2 |
| E-04 | 周期性调度 | 简单模式：每 N 分钟/小时/天/周/月执行 | **S** | 2 |
| E-05 | 调度启用/禁用 | 快速暂停和恢复定时调度，不删除配置 | **S** | 2 |
| E-06 | 依赖触发 | 上游执行成功后自动触发下游 | **S** | 2 |
| E-07 | 重试策略 | 配置失败后自动重试次数和退避策略 | **S** | 2 |
| E-08 | 执行参数注入 | 支持传入参数变量（如当前日期 `{today}`） | **N** | 3 |
| E-09 | 事件触发 | 监听 SFTP 文件到达事件自动触发执行 | **N** | 3 |

## 模块 F — 监控与告警 (Monitoring & Alerting)

| 功能 ID | 功能名称 | 描述 | 优先级 |
|---------|---------|------|-------|
| F-01 | 执行历史列表 | 查看所有管道的历史执行记录，支持过滤和搜索 | **M** |
| F-02 | 执行详情与日志 | 每次执行的详细日志、节点耗时、处理行数统计 | **M** |
| F-03 | 实时执行状态 | 正在运行中的管道节点级别进度展示 | **S** |
| F-04 | 仪表盘 (Dashboard) | 汇总展示管道健康状态、成功率、近期执行趋势 | **S** |
| F-05 | 失败告警 — 邮件 | 执行失败时自动发送邮件通知至指定收件人 | **M** |
| F-06 | 失败告警 — Webhook | 发送失败通知到 Slack / Teams / 钉钉等 | **S** |
| F-07 | 数据质量报告 | 按管道维度展示数据验证通过率历史趋势 | **S** |
| F-08 | 性能指标 | 执行时长、吞吐量（行/秒）趋势图 | **N** |
| F-09 | 错误样本查看 | 查看数据验证失败的具体数据行样本 | **S** |
| F-10 | 操作审计日志 | 记录用户操作行为（创建/修改/删除/执行），保留 90 天 | **S** |
