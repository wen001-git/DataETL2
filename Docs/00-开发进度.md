## 已确认决策记录

| 决策项 | 结论 | 确认时间 |
|--------|------|---------|
| 技术栈 | React 18 + FastAPI + MySQL + DuckDB + paramiko + Docker Compose | 2026-05-02 |
| MVP 范围 | Phase 1 做全链路 Raw→DWD→DWS→ADS | 2026-05-03 |
| MySQL 部署 | Docker Compose 容器内新建，不依赖外部数据库 | 2026-05-03 |
| 零硬编码原则 | 所有表名/字段名/数据源名均由用户配置，系统不预设业务逻辑 | 2026-05-02 |
| Phase 1 UI | 表单式 UI（非拖拽画布），React Flow 推迟到 Phase 2 | 2026-05-03 |
| Raw 数据保留 | etl_raw 层数据永久保留，支持追溯和重跑 | 2026-05-03 |
