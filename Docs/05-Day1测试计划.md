## TC-06 加密模块 (`crypto.py`)

在容器内执行 Python 验证：

```bash
docker compose exec api python3 -c "
from crypto import encrypt, decrypt

# TC-06-1: 正常加解密
msg = 'sftp_secret_password_123'
enc = encrypt(msg)
dec = decrypt(enc)
assert dec == msg, f'FAIL: {dec} != {msg}'
print('TC-06-1 PASS: encrypt/decrypt roundtrip OK')

# TC-06-2: 空字符串安全处理
assert encrypt('') == '', 'FAIL: empty encrypt'
assert decrypt('') == '', 'FAIL: empty decrypt'
print('TC-06-2 PASS: empty string handled')

# TC-06-3: 加密结果不含明文
enc2 = encrypt('mysecret')
assert 'mysecret' not in enc2, 'FAIL: plaintext visible in ciphertext'
print('TC-06-3 PASS: ciphertext does not expose plaintext')

print('ALL crypto tests PASSED')
"
```

| # | 测试项 | 预期结果 |
|---|--------|---------|
| 6.1 | 加解密往返一致 | `decrypt(encrypt(x)) == x` |
| 6.2 | 空字符串安全处理 | `encrypt("") == ""`, `decrypt("") == ""` 不抛异常 |
| 6.3 | 密文不含明文 | 密文字符串中不包含原始密码 |
